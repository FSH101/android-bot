# Windows-сервер open.mp с поддержкой Node.js

В этой директории находится вспомогательный скрипт, который собирает готовый сервер [open.mp](https://github.com/openmultiplayer/open.mp) для Windows, подключает компонент `omp-node.dll` и разворачивает тестовый гейммод на JavaScript.

## Требования

* Linux c установленными `curl`, `unzip`, `npm` и `python3`.
* Доступ в интернет для загрузки официального релиза сервера open.mp и компонента `omp-node.dll`.
* Установленный Node.js (необходим для установки зависимостей).

## Использование

```bash
./scripts/setup_openmp_server.sh
```

Скрипт скачивает актуальную Windows-сборку сервера open.mp, компонент `omp-node.dll`, устанавливает пакет `@open.mp/node`, создаёт `server.json` и гейммод `gamemodes/gamemode.js`, который выводит сообщение «Сервер нейросети запущен!» в консоль и игровую чат-панель.

После завершения работы скрипта все файлы находятся в директории `runtime/windows/openmp`. Скопируйте её на машину с Windows (или смонтируйте в WSL) и запустите сервер командой:

```bash
cd "C:\\путь\\к\\openmp"
.\\omp-server.exe
```

При старте сервер загрузит Node.js-гейммод и отправит приветственное сообщение всем подключённым игрокам. Конфигурация задаётся в файле `server.json`.

## Очистка

Удалите директорию `runtime`, чтобы убрать все сгенерированные файлы.
